# Azraqmart: منصة التجارة الإلكترونية B2B المتكاملة

![Azraqmart Logo](https://placehold.co/1200x600/007bff/ffffff?text=Azraqmart+B2B+Platform)

## نظرة عامة

Azraqmart هي منصة تجارة إلكترونية متكاملة مصممة خصيصًا لتلبية احتياجات الشركات (B2B) في السوق العربي. توفر المنصة حلاً شاملاً لتجار الجملة، يغطي جميع جوانب العمليات التجارية من إدارة المنتجات والطلبات إلى المحاسبة والتوصيل، مع التركيز على تجربة مستخدم سلسة وفعالة.

## الميزات الرئيسية

تم تصميم Azraqmart بمجموعة واسعة من الميزات القوية لدعم عمليات B2B المعقدة:

*   **لوحة تحكم إدارية شاملة**: لإدارة المستخدمين، المنتجات، الطلبات، والتقارير.
*   **كتالوج منتجات متقدم**: مع فلاتر بحث وتصنيفات هرمية.
*   **إدارة الطلبات**: تتبع حالة الطلبات من الإنشاء حتى التسليم.
*   **نظام إدارة المخزون**: لتتبع الوارد والصادر، الجرد، وتنبيهات نفاد الكمية.
*   **نظام محاسبي متكامل**: لإدارة المديونيات، الفواتير، والتقارير المالية.
*   **نظام توصيل فعال**: مع تتبع لحظي للمندوبين وتعيين تلقائي للطلبات.
*   **إدارة العملاء والموظفين**: مع نظام صلاحيات دقيق.
*   **إشعارات Push**: لتحديثات الطلبات والعروض.
*   **شات بوت ذكي**: للرد على استفسارات العملاء تلقائيًا.
*   **دعم متعدد المنصات**: تطبيق ويب متجاوب وتطبيق جوال أصلي (Android/iOS عبر Capacitor).
*   **تصميم عربي أصيل (RTL)**: مع دعم كامل للغة العربية.

## التقنيات المستخدمة

تم بناء Azraqmart باستخدام أحدث التقنيات لضمان الأداء العالي، قابلية التوسع، وسهولة الصيانة:

*   **الواجهة الأمامية (Frontend)**:
    *   **TypeScript**: لغة برمجة قوية وموسعة لـ JavaScript.
    *   **React 19**: مكتبة JavaScript لبناء واجهات المستخدم التفاعلية.
    *   **Tanstack Router & React Query**: لإدارة التوجيه وحالة البيانات.
    *   **Vite**: أداة بناء سريعة لتجربة تطوير محسّنة.
    *   **Tailwind CSS v4**: إطار عمل CSS قائم على الأدوات المساعدة لتصميم سريع وجذاب.
    *   **Radix UI**: مكونات واجهة مستخدم عالية الجودة.

*   **الواجهة الخلفية وقاعدة البيانات (Backend & Database)**:
    *   **Supabase**: منصة مفتوحة المصدر توفر قاعدة بيانات PostgreSQL، مصادقة، تخزين، ووظائف فورية.
    *   **Cloudflare Workers**: للحوسبة بلا خادم (serverless) وأداء عالي.
    *   **PLpgSQL**: للوظائف والمشغلات في قاعدة البيانات.

*   **تطبيقات الجوال (Mobile Applications)**:
    *   **Capacitor**: لتحويل تطبيق الويب إلى تطبيقات جوال أصلية (Android و iOS).
    *   **PWA (Progressive Web App)**: لتقديم تجربة تطبيق أصيل عبر الويب.

## البنية المعمارية

تعتمد Azraqmart على بنية معمارية حديثة قائمة على الخدمات المصغرة (Microservices) والوظائف بلا خادم (Serverless Functions) لضمان المرونة وقابلية التوسع. يتم استخدام Supabase كحل شامل للواجهة الخلفية، بينما يتم نشر الواجهة الأمامية والوظائف على Cloudflare Workers لتحقيق أقصى قدر من الأداء والكفاءة.

## قابلية النشر والتوسع

تم تصميم Azraqmart ليكون قابلاً للنشر بسهولة والتوسع تلقائيًا للتعامل مع أحمال العمل المتزايدة. يضمن استخدام Cloudflare Workers و Supabase قدرة المنصة على خدمة عدد كبير من المستخدمين والمعاملات بكفاءة عالية.
